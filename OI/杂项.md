# 杂项整理

## 随机化算法

```cpp
LL randint(LL l, LL r)
{
    static std::mt19937 eng(std::random_device{}());
    static std::unifor_int_distribution<LL> dis(l, r);
    return dis(eng);
}
```

## O(1) 快速乘

```cpp
LL mul(LL a, LL b, LL m)
{
    LL r = ((long double)a / m * b + 0.5);
    r = a * b - r * m;
    return r < 0 ? r + m : r;
}
```
